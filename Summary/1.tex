	\subsection{存储区域}
		计算机有5个存储区域:
	\begin{itemize}
	\item 堆，程序员自己申请的内存储存的区域
	\item 栈，编译器管理的区域，不用时自动销毁
	\item 全局静态区，全局变量，static关键字修饰的变量
	\item 常量区，const修饰的变量 存储的区域，不可修改
	\item 代码区 函数的二进制储存区
	\end{itemize}
